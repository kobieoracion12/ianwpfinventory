Pushing Existing Project to a Repository

1. Git Bash on your directory/folder

2. git init
This is used to make your folder a working directory for git

3. git status

4. git add .

5. git commit -m "COMMENT"

6. Create a new Repository without README.md

7. git remote add origin <URL>

8. git push origin master

9. git pull origin master


Change remote origin URL

git remote set-url origin <URL>
